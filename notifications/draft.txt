8. Добавить бота по имени к чату и делаем его админом
9. Вставляем токен в CURL

13. Скачиваем allure-notification.jar https://github.com/qa-guru/allure-notifications/releases и добавляем в проект
14. Создаем файл config.json (его лучше не отправлять в репозиторий, т.к. там указан токен)
15. При запуске в Jenkins добавить в секцию build первым пунктом файл config.json
16. И в post build task SCRIPT java "-DconfigFile=notifications/config.json" -jar notifications/allure-notifications-4.2.1.jar
